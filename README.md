# NorthStar Urban Mobility Analytics

## Assignment Overview
This repository contains the complete analysis for the NorthStar Urban Mobility case study, completed as part of the Databases and Analytics module (CP60056E).

## Files Included
- `NorthStar_Complete_Analysis.ipynb` - Main Jupyter notebook with all Python, MongoDB, and R code
- `customer_journeys.json` - Nested document structure for NoSQL/MongoDB section

## Analysis Performed
1. **Python Data Processing** - Data cleaning, missing value handling, zone performance analysis
2. **MongoDB/NoSQL** - Nested document design for app_events data
3. **R + SQL** - Complaint analysis, zone performance, driver analysis
4. **Query Optimization** - Index comparison demonstrating performance improvement

## Key Findings
- Most common complaint type: **Delay** (101 cases)
- Average complaint resolution time: **8.3 days**
- Total compensation paid: **£6,158**
